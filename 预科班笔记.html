CSS

CSS(Cascading Style Sheets)

CSS通常称为CSS样式表或层叠样式表（级联样式表），主要用于设置HTML页面中的文本内容（字体、大小、对齐方式等）、图片的外形（宽高、边框样式、边距等）以及版面的布局等外观显示样式。

  • 基本格式

    <style type="text/css">
    p {
        color: red;
        font-size: 10px;
    }
</style>

注意位置: 写在Head标签中

  • 字体

    p {
   font-family: "微软雅黑", arial;
}

  • 加粗

    font-weight: 700;
//700相当于bold; 400相当于normal

  • ID选择器

id选择器使用“#”进行标识，后面紧跟id名

#pink {
    color: pink;
}

<p id="pink">ID选择器的使用</p>

W3C标准规定，在同一个页面内，不允许有相同名字的id对象出现，但是允许相同名字的class。

类选择器（class） 好比人的名字， 是可以多次重复使用的， 比如 张伟 王伟 李伟 李娜

id选择器 好比人的身份证号码， 全中国是唯一的， 不得重复。 只能使用一次。

  • 颜色的三种格式

    1.预定义的颜色值，如red，green，blue等。

2.十六进制，如#FF0000，#FF6600，#29D794等。实际工作中，十六进制是最常用的定义颜色的方式。

3.RGB代码，如红色可以表示为rgb(255,0,0)或rgb(100%,0%,0%)。

  • Js的书写格式

    一行书写一个语句，并且用分号表示语句结束

  • Js的注释

    单行注释：//      快捷键： ctrl+/

多行注释：/**/      快捷键： ctrl+shift+/

  • 几种输出语句的方式

    • alert 弹窗

      alert("Hello World!!!");

    • prompt 弹窗带文本框

      prompt("你好呀!", "我很好!");

    • console 控制台输出语句

      console.log("你好呀!!!");

    • document.write 文档打印输出

      document.write("你好呀!!!");

  • 什么是变量?

    姓名: 张三
name: zhangsan
var name = "zhangsan";

    • 命名方式

      1.变量命名必须以字母或是下标符号”_”或者”$”为开头。

2.变量名长度不能超过255个字符。

3.变量名中不允许使用空格。

4.不用使用脚本语言中保留的关键字及保留符号作为变量名。

5.变量名区分大小写。(javascript是区分大小写的语言)

    • 练习输入名字

username = prompt("请输入你的名字");
document.write(""您的名字是:" + username);

    • 数值型 number

      var num = 10;

    • 字符型 string

      var name = "张三"; var job = '程序员';

    • 布尔型 boolean

      var flag = true;

    • 未定义类型 undefined

    • 空型 null 清空变量
